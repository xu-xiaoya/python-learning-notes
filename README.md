# Python Learning Notes

这是一个按天记录的 Python 基础练习项目，目标是把“语法点 + 小例子 + 自己的理解”逐步积累下来，方便后续复习。

## 当前结构

```text
python/
|- day1.py       # Day 1：数据类型与基础操作
|- day2.py       # Day 2：条件判断、循环、输入与列表处理
|- day1_test.py  # 一些额外的小练习
`- README.md     # 项目说明
```

## 这样分组的思路

我把内容按“先认识数据，再学习控制流程”重新整理了：

1. 先学数据类型和基础操作
2. 再学条件判断 `if`
3. 再把 `for`、`while`、`range` 放到一起
4. 最后把 `input()` 和列表处理练习放在循环后面

这样的顺序更适合复习，因为知识点之间的依赖更自然：

- 先知道“要操作什么数据”
- 再知道“如何判断”
- 然后学习“如何重复执行”
- 最后做带交互的小练习

## 每个文件在做什么

### `day1.py`

按“数据与操作”的顺序整理，主要包含：

- 字符串
- 数字与运算
- 列表
- 排序
- 切片与复制
- 元组
- 字典与集合

适合用来复习 Python 的基础数据类型。

### `day2.py`

按“流程控制”的顺序整理，主要包含：

- 条件判断 `if`
- `for` 循环和 `range`
- 循环中的条件判断
- 输入 `input()`
- `while` 循环
- 列表处理练习

适合用来复习“程序是怎么控制执行流程的”。

## 如何运行


运行 Day 2：

```bash

```
