# Python Learning Notes

这是一个按天记录的 Python 基础练习项目，目标是把“语法点 + 小例子 + 自己的理解”逐步积累下来，方便后续复习。

## 当前结构

```text
python/
|- day1.py        # Day 1：数据类型与基础操作
|- day1_test.py   # 一些额外的小练习
|- day2.py        # Day 2：条件判断、循环、输入与列表处理
|- day3.py        # Day 3：函数、默认参数、任意参数
|- day3_import.py # Day 3：模块导入示例
|- day4/          # Day 4：类、继承、组合，已拆成文件夹
|  |- __init__.py
|  |- __main__.py
|  |- dog.py
|  |- car.py
|  |- electric_car.py
|  `- demo.py
|- day4_import.py # Day 4：类导入示例
`- README.md      # 项目说明
```

## 学习顺序

我把内容按“先认识数据，再控制流程，再组织代码，最后进入面向对象”的顺序整理：

1. 数据类型与基础操作
2. 条件判断和循环
3. 函数与模块导入
4. 类与对象

先学会“数据怎么用”，再学会“逻辑怎么跑”，然后再学“代码怎么组织”

## 每个文件在做什么

### `day1.py`

按“数据与操作”的顺序整理，主要包含：

- 字符串
- 数字与运算
- 列表
- 排序
- 切片与复制
- 元组
- 字典与集合

适合用来复习 Python 的基础数据类型。

### `day2.py`

按“流程控制”的顺序整理，主要包含：

- 条件判断 `if`
- `for` 循环和 `range`
- 循环中的条件判断
- 输入 `input()`
- `while` 循环
- 列表处理练习

适合用来复习“程序是怎么控制执行流程的”。

### `day3.py`

主要整理函数相关内容，包含：

- 定义函数 `def`
- 默认参数
- 位置参数和关键字参数
- 任意数量参数 `*args`

适合用来复习“如何把重复逻辑封装成函数”。

### `day3_import.py`

这是一个简单的模块导入示例，演示：

- 如何从其他文件中导入函数
- 如何使用 `as` 给导入函数起别名

适合配合 `day3.py` 一起看。

### `day4/`

这是 Day 4 的类学习文件夹，主要包含：

- `dog.py`：最基础的类和对象
- `car.py`：属性读取和修改
- `electric_car.py`：继承、重写父类方法、组合
- `demo.py`：统一演示入口
- `__main__.py`：支持直接运行整个 Day 4 包

适合作为面向对象入门。

### `day4_import.py`

这是 Day 4 的导入示例，演示如何从 `day4/` 文件夹中导入类。

## 如何运行

运行 Day 1：

```bash
python day1.py
```

运行 Day 2：

```bash
python day2.py
```

运行 Day 3：

```bash
python day3.py
python day3_import.py
```

运行 Day 4：

```bash
python -m day4
python day4_import.py
```

## 后续建议

- 继续保持“一个主题一个函数”或“一个主题一个类”的写法
- 每个文件都保留 `main()` 作为统一入口
- 学到类之后，可以慢慢练习把“数据 + 行为”放进同一个类里
- 如果后面内容越来越多，可以再拆成 `notes/`、`examples/`、`exercises/`
